## Q. 문자열을 거꾸로 읽어도 동일한지 여부를 재귀적으로 결정하기

### 문제

문자열을 거꾸로 읽어도 동일한지 여부를 재귀적으로 결정하기 위한 프로그램을 어떤 식으로 만들 것인가 ?

---

 <br/>

> ### 풀이

```javascript
function checkReadBackword(arr) {
  let backwordStr = '';
  for (let i = arr.length - 1; i > -1; i--) {
    backwordStr += arr[i];
  }
  if (backwordStr === arr) {
    return true;
  } else {
    return false;
  }
}
```

> ### 입력

```javascript
'testset';
```

> ### 출력

```javascript
true;
```
